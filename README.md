## How to Use

### 1. Add Images
Place your restaurant images in the `images/` folder:
- `logo.png` - Restaurant logo
- `slide1-01.jpg`, `master-slides-01.jpg`, etc. - Hero slider images
- `our-story-01.jpg` - About section image
- `intro-01.jpg`, `intro-02.jpg`, `intro-04.jpg` - Feature images
- `our-menu-*.jpg` - Menu category images
- `photo-gallery-*.jpg` - Gallery images
- `event-*.jpg` - Event images
- `team-*.jpg` - Team member photos
- `blog-*.jpg` - Blog post images

### 3. Add Fonts (Optional)

- Place font files in the `fonts/` folder
- Update `css/main.css` to reference them

### 4. Test the Website
Open `index.html` in a web browser to test:
- Navigation between pages
- Responsive design (test on mobile)
- Form submissions
- Image loading
