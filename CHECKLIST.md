## 💡 Quick Tips

1. **Image Sizes**: Recommend using compressed images for web (200-500KB each)
2. **File Names**: Keep lowercase, no spaces
3. **Testing**: Use Firefox or Chrome developer tools to test responsive design
4. **Backup**: Keep a backup of your files
5. **Color Scheme**: Gold (#d4a574) and Dark backgrounds work well for restaurants

## ❓ Common Issues & Solutions

**Issue**: Images don't show up
**Solution**: Make sure image files are in the `images/` folder with exact names as referenced in HTML

**Issue**: Sidebar doesn't toggle
**Solution**: Check if JavaScript files loaded (open browser console)

**Issue**: Forms don't submit
**Solution**: Forms require backend setup - currently they show a confirmation message locally

**Issue**: Responsive design looks wrong
**Solution**: Clear browser cache (Ctrl+F5) and test in incognito mode
